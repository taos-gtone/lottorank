2) DB 연결 / 프로그래밍이 필요한 부분

인증 (Auth)

[ 로그인 폼 ]  →  회원 DB 조회 + 세션/토큰 처리
[ 네이버 / 카카오 소셜 로그인 ]  →  OAuth 2.0 연동
[ 회원가입 ]  →  회원 DB INSERT
실시간 데이터 (DB 조회)

[ 최근 당첨번호 - 제1161회 ]
  ↳ lotto_result 테이블에서 최근 회차 번호 7개 + 보너스 SELECT

[ 성적표 - 1등/2등/3등 당첨자수 / 당첨금 ]
  ↳ lotto_prize 테이블에서 회차별 등위 데이터 SELECT

[ 히어로 통계 숫자 ]
  - 1,162 진행 회차  →  현재 회차 번호
  - 23,847 누적 예측자  →  user 테이블 COUNT
  - 72.4% TOP1 적중률  →  ranking 테이블에서 1위 사용자 적중률
  - 3,120 이번 주 예측  →  이번 회차 predict 테이블 COUNT
랭킹 (DB 조회)

[ 전체 랭킹 테이블 (이번 주 / 누적 전체 / 이달의 랭킹 탭) ]
  ↳ predict + result JOIN → 사용자별 적중률 계산 → 순위 정렬
  ↳ 탭 전환 시 각각 다른 쿼리 (이번 회차 / 전체 / 이달)

[ TOP 3 카드 - 닉네임 / 연속 참여주 / 적중률 ]
  ↳ ranking 테이블 TOP 3 SELECT

[ 순위 변동 (▲▼-) ]
  ↳ 직전 회차 ranking vs 현재 ranking 비교
예측 제출 (DB INSERT)

[ 번호 예측 제출 버튼 ]
  ↳ predict 테이블 INSERT (user_id, round, number, created_at)
  ↳ 중복 제출 방지 (이미 제출 여부 확인)
  ↳ 마감 시간 체크 (토요일 오전 8시)
포인트 / 결제 (DB 조회 + 차감)

[ 🔓 열람 (500P) 버튼 - TOP 3 예측번호 잠금 해제 ]
  ↳ 사용자 포인트 잔액 확인
  ↳ 포인트 차감 → predict 번호 공개
  
[ 골드 멤버십 가입하기 ]
  ↳ 결제 연동 + membership 테이블 UPDATE
사이드바 (로그인 상태 연동)

[ 골드 멤버십 - 135명 실제 1등 배출 ]
  ↳ 통계 DB 조회

[ 빠른 통계 - 490회 / 135회 / 1위 ]
  ↳ 집계 테이블 SELECT
